
# MySQL Employee Database Queries – Day 2

## Project Description
This project contains SQL queries performed on an Employee database for practicing MySQL concepts.  
The main objective of this task is to improve understanding of data retrieval, filtering, grouping, and searching techniques in SQL.

---

## Technologies Used

- MySQL 5.7
- DB Fiddle / MySQL Workbench

---

## SQL Concepts Implemented

### 1. SELECT Queries
Used to retrieve records from the Employees table.

### 2. WHERE Clause
Used for filtering rows based on conditions.

### 3. GROUP BY Clause
Used to group rows having similar values.

### 4. HAVING Clause
Used to filter grouped data.

### 5. DISTINCT Keyword
Used to display unique values.

### 6. ORDER BY and LIMIT
Used to sort and display top records.

### 7. Operators Used
- Comparison Operators
- Logical Operators
- BETWEEN
- IN / NOT IN
- LIKE Operator

---

## Example Operations Performed

- Display employees from IT department
- Find employees with salary greater than 70000
- Count employees city-wise
- Find maximum salary department-wise
- Retrieve top highest paid employees
- Search employee names using LIKE patterns

---
## Outcomes

This practice helped in understanding:

- SQL query execution
- Data filtering techniques
- Aggregate functions
- Pattern matching
- Data grouping and analysis
