# JOIN Operations

## Description

This assignment focuses on SQL JOIN operations and relationship handling between database tables in an online learning platform dataset.

## Database Tables Used

- students
- courses
- enrollments
- instructors

## Implemented Queries

1. Student and course enrollment report
2. Courses without enrollments
3. Instructor and course mapping
4. Courses without instructors
5. RIGHT JOIN examples
6. Students without enrollments
7. FULL OUTER JOIN implementation
8. Unused courses report
9. Instructor-course unmatched data
10. Combined student-course-instructor report
11. CROSS JOIN challenge

## Concepts Learned

- Table relationships
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- CROSS JOIN
- UNION operations
- Handling NULL values

## Result

Successfully executed all JOIN-based SQL queries and understood how different join operations work in relational databases.
